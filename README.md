# Vehicle Management System
Acest branch conține rezolvarea pentru exercițiul de gestiune a vehiculelor.

**Concepte folosite:**
* Programare Orientată pe Obiecte (OOP) - Încapsulare, Clase Abstracte, Moștenire
* Polimorfism (implementări specifice pentru `Car`, `Motorcycle`, `Truck`)
* Gestiunea colecțiilor folosind array-uri (clasa `Garage`)
* Suprascrierea metodelor de bază (ex: `.equals()` pentru compararea mașinilor)
