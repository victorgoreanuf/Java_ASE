# Java Reflection - Exerciții
Acest branch conține rezolvarea a 11 exerciții practice ce utilizează API-ul de Reflection din Java.

**Capabilități implementate:**
* Inspecția claselor: Metode, Variabile, Tipuri de date și Modificatori de acces.
* Manipulare la Runtime: Instanțiere dinamică prin mai mulți constructori.
* Acces de nivel profund: Citirea și modificarea câmpurilor/metodelor `private` folosind `setAccessible(true)`.
* **Proiecte practice (Intermediar):**
  1. **Object Inspector** - Citirea automată a stării unui obiect la runtime.
  2. **JSON Serializer** - Conversia obiectelor Java într-un format JSON valid folosind exclusiv Reflection.
  3. **CSV Mapper** - Maparea dinamică a unui rând de tip CSV direct într-o instanță de obiect, detectând automat tipurile de date.
