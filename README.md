# Bank Account System
Acest branch conține rezolvarea pentru exercițiul cu sistemul bancar.

**Concepte folosite:**
* Programare Orientată pe Obiecte (OOP) - Încapsulare, Moștenire (SavingsAccount)
* Tratarea excepțiilor personalizate (`InvalidAmountException`, `InsufficientFundsException`)
* Validarea operațiunilor (depuneri, retrageri, sold negativ)
