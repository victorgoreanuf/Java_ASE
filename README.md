# Exerciții Colecții Java (Collections Framework)
Acest branch conține o serie de exerciții practice pentru aprofundarea structurilor de date în Java.

**Concepte acoperite:**
* **List / ArrayList:** Manipulare, iterare, sume, inversare manuală, sortare folosind `Comparator`.
* **Set / HashSet / LinkedHashSet:** Identificarea elementelor unice, filtrarea duplicatelor de obiecte personalizate.
* **Map / HashMap:** Numărarea frecvenței cuvintelor, mapări cheie-valoare (tip Phone Book), combinarea map-urilor folosind `merge`.
* **LinkedHashMap:** Implementarea simplificată a unui **LRU Cache** (Least Recently Used) suprascriind `removeEldestEntry`.
* **OOP & Colecții:** Suprascrierea metodelor `.equals()` și `.hashCode()` pentru logica de filtrare a duplicatelor din listele de obiecte customizate.
